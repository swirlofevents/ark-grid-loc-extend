# Калькулятор Арк Грида

Легковесный веб-инструмент для оптимизации раскладки рунитов в ядрах Арк Пассива на основе вашего инвентаря.


## Как это работает

Калькулятор симулирует все возможные комбинации рунитов для 3 ядер на основе:

- Ваших доступных рунитов (инвентарь классовых / общих рунитов)
- Раритета ядра (Легендарное / Реликвия / Древнее)
- Лимитов заряда для каждого ядра
- Целевых значений очков (Авто или Вручную)

Затем выбирается лучшая возможная комбинация по следующим критериям:

- Приоритет достижения целевых очков
- Минимизация потраченных впустую очков (превышение цели)
- Максимизация общего количества очков
- Использование наименьшего заряда при равных условиях


## Как использовать

### 1. Откройте страницу Арк Грида
- Заполните количество каждого рунита, который у вас есть.
- Классовые и общие руниты рассчитываются отдельно.

### 2. Настройте ядра
- Выберите **Легендарное / Реликвия / Древнее** для каждого ядра.
- **Автоматический выбор цели (рекомендуется):**
  - Легендарное → 14 очков
  - Реликвия / Древнее → 17 очков
- Или отключите автоматический выбор цели и установите цели вручную.

### 3. Рассчитайте
- Нажмите **Рассчитать** для оптимизации одной стороны.
- Или **Рассчитать Оба** для одновременной оптимизации классовых и общих рунитов.

### 4. Прочитайте результаты
Для каждого ядра вы увидите:
- Использованные руниты
- Использованный заряд
- Очки относительно цели
- Полосу прогресса
- Оставшиеся неиспользованные руниты

### 5. Предложение по добавлению / созданию (достичь всех целей)
Если ваш текущий инвентарь **не может полностью достичь всех целевых очков**, калькулятор покажет дополнительную секцию:

**Добавить / Создать Предложение (достичь всех целей)**

Эта функция:
- Автоматически проверяет, что произойдет, если вы **добавите или создадите дополнительные руниты**
- Находит **минимальное количество дополнительных рунитов**, необходимых
- Предлагает тип рунита (или типы), который гарантирует, что **все ядра достигнут своих целей**
- Ищет до **8 дополнительных рунитов** с использованием стратегии минимального добавления

Предложение представляет:
- **Наихудший безопасный вариант**
- Добавление указанных рунитов гарантирует достижение всех целей, даже если другие варианты также могут сработать

Пример:
- `A × 1` означает, что создание или добавление **одного рунита ранга A** достаточно для достижения всех целей.


## Примечания
- Результаты всегда учитывают лимиты вашего инвентаря.
- Если инвентарь недостаточен, калькулятор показывает лучший достижимый результат ниже цели.
- Ручные цели влияют на порядок приоритета между ядрами.
- Предложения по добавлению / созданию **не изменяют** ваш инвентарь — они носят информационный характер.
- Сайт может продолжать получать обновления и улучшения со временем.
